# TiendaService

Proyecto Maven de ejemplo que contiene las clases `Producto`, `Cliente`, `Venta` y la clase principal `TiendaService` con `main`.

Compilar y ejecutar:

```bash
mvn -q package
java -cp target/TiendaService-1.0-SNAPSHOT.jar com.tienda.service.TiendaService
```

Nota: requiere Java 17 y Maven instalados.
